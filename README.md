MuseoSpace reimagines an art gallery as a four-stage digital journey, each crafted for a specific visual and interactive experience. Each section demonstrates creative front-end mastery — animations, dynamic DOM changes, and responsive layouts — within a fully client-side environment.
 What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
